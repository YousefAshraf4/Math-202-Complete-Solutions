6 Prove A is orthogonal and find A⁻¹
A = ⎡ 1/2 √3/2 0⎤ ⎢−√3/2 1/2 0⎥ ⎣ 0 0 1⎦
A matrix is orthogonal iff AAᵗ = I, and then A⁻¹ = Aᵗ.
  • Write Aᵗ: Aᵗ = ⎡ 1/2 −√3/2 0⎤ ⎢ √3/2 1/2 0⎥ ⎣ 0 0 1⎦
  • Compute AAᵗ (the rows of A dotted with the cols of Aᵗ, which are the rows of A): Row1·Col1: (1/2)² + (√3/2)² + 0 = 1/4 + 3/4 = 1 ✓ Row1·Col2: (1/2)(−√3/2) + (√3/2)(1/2) + 0 = −√3/4 + √3/4 = 0 ✓ Row1·Col3: 0 ✓ Row2·Col1: (−√3/2)(1/2) + (1/2)(√3/2) + 0 = 0 ✓ Row2·Col2: (−√3/2)² + (1/2)² + 0 = 3/4 + 1/4 = 1 ✓ Row2·Col3: 0 ✓ Row3·Col3: 0² + 0² + 1² = 1 ✓
  • Therefore AAᵗ = I → A is orthogonal.
A is orthogonal; A⁻¹ = Aᵗ = ⎡ 1/2 −√3/2 0⎤ ⎢ √3/2 1/2 0⎥ ⎣ 0 0 1⎦
7 A = (1/√2)[[1],[−1]] — prove B = I − 2AAᵗ is orthogonal
  • Compute AAᵗ (column vector × row vector): AAᵗ = (1/√2)⎡1⎤·(1/√2)[1 −1] = (1/2)⎡ 1 −1⎤ ⎣−1⎦ ⎣−1 1⎦
  • Compute 2AAᵗ and then B: 2AAᵗ = ⎡ 1 −1⎤ ⎣−1 1⎦ B = I − 2AAᵗ = ⎡1 0⎤ − ⎡ 1 −1⎤ = ⎡0 1⎤ ⎣0 1⎦ ⎣−1 1⎦ ⎣1 0⎦
  • Check BBᵗ: Bᵗ = B (B is symmetric), so BBᵗ = B²: B² = ⎡0 1⎤·⎡0 1⎤ = ⎡1 0⎤ = I ✓ ⎣1 0⎦ ⎣1 0⎦ ⎣0 1⎦
BBᵗ = I → B is orthogonal. (B is a reflection matrix.)
8 Prove: the determinant of any orthogonal matrix equals ±1
  • Let A be orthogonal, so by definition: AAᵗ = I
  • Take the determinant of both sides: |AAᵗ| = |I| = 1
  • Use the multiplicative property of determinants and |Aᵗ| = |A|: |A|·|Aᵗ| = 1 → |A|·|A| = 1 → |A|² = 1
  • Therefore: |A| = ±1 □
+1 corresponds to a rotation; −1 corresponds to an improper rotation (rotation + reflection).
